Daniel Berenberg's Personal Site

About me

My name is Daniel Berenberg. I am a 23 year old graduate student pursuing an M.S in Computer Science at the University of Vermont.

My academics focus on complex systems and data science: applying artificial intelligence and network analytics to deep and fascinating problems in both the text and image domains.

My academic advisor is Dr. James Bagrow. We mine large textual datasets called human knowledge graphs and use state of the art computational linguistics theory and tools to research complex interactions such as causal relationships.

I enjoy implementing high performance scientific computing software and integrating cloud GPU and cluster services to build robust and efficient data analytics algorithms as well as computation.

I am also a member of VAiL, the Vermont Artificial Intelligence Laboratory, directed by Dr. Safwan Wshah. There I am a machine learning engineer researching applications for 3D convolutional neural networks in the biomedical domain.

Outside of my work, I can be found running, hiking, skateboarding, or climbing.

Projects

Some selected projects that I have either completed or are currently underway.

  • WePanic (VAiL)

    Building a deep learning system to predict user heart rate and respiratory rate from a smart phone video with super human accuracy. Implementing 3D convolutional architectures and video processing algorithms with various Python scienctific computing libraries such as scipy, numpy, opencv, and Keras

    .

  • Graph++ (Spring 2018)

    A C++ implementation of generalized network algorithms. Developed for a Spring 2018 course called Algorithms for Complex Networks. Efficiently implemented a suite of network analysis tools for large networks.

  • Playing Asteroids with Deep Reinforcement Learning (Fall 2017)

    Third place recipient in UVM 2017 Computer Science fair. Implemented an artificially intelligent reinforcement learning agent that and trained it to play a reduced difficulty version of a homebrewed version of the classic Asteroids game. Asteroids game was implemented in C++, AI system was implemented using Tensorflow and other selected python libraries. Final DNN architecture was a modified double dueling Deep Q-Network, a tested convolutional architecture that uses the Q-learning algorithm.
